## 学习路线

1. **基础数据结构**：数组 → 链表 → 栈/队列 → 哈希表
2. **基础算法**：排序 → 二分查找 → 递归/回溯
3. **进阶数据结构**：树 → 堆 → 图
4. **进阶算法**：动态规划 → 贪心 → 滑动窗口
5. **刷题实战**：按分类刷 LeetCode
