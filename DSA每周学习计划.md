# 数据结构与算法每周学习计划

依据：《数据结构与算法：Python 语言实现》

建议周期：12 周  
建议频率：每周 5 天学习，2 天复盘/补缺/休息  
建议时长：每天 60-90 分钟

## 第 1 周：Python 基础与面向对象

对应章节：
- 第 1 章 Python 入门
- 第 2 章 面向对象编程

本周目标：
- 熟悉 Python 中对象、函数、控制流程、异常、迭代器、生成器
- 能定义类、实例方法、特殊方法和简单继承
- 为后续手写数据结构做语言准备

每日安排：
- 周一：阅读 1.1-1.3，复习对象、变量、表达式、运算符
- 周二：阅读 1.4-1.7，复习条件、循环、函数、异常
- 周三：阅读 1.8-1.11，理解迭代器、生成器、模块导入
- 周四：阅读 2.1-2.3，学习类定义、封装、特殊方法
- 周五：阅读 2.4-2.6，学习继承、命名空间、浅拷贝和深拷贝
- 周六：补做代码任务和习题
- 周日：复盘本周知识点

代码任务：
- 实现 `find_max(data)`
- 实现 `is_even(k)`
- 实现 `Vector` 类，支持加法、减法、取长度、索引访问
- 实现一个简单迭代器类

练习重点：
- 第 1 章练习：选择 6 道
- 第 2 章练习：选择 6 道

验收标准：
- 能解释 Python 中“变量名引用对象”的含义
- 能写出一个带 `__len__`、`__getitem__`、`__str__` 的类
- 能区分浅拷贝和深拷贝
